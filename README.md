# SISFOKOL
SISFOKOL (SIStem inFOrmasi seKOLah), aplikasi basis web dan basis android, untuk keperluan manajemen sekolah. Mulai dari akademik, penilaian, raport dan seterusnya...

Script : PHP + Mysql + Jquery + Phonegap Cordova
